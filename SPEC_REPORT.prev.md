# Отчёт по итерации

**Версия:** 0.31.0
**Дата:** 2026-04-02

## Выполнено

### Must Have 1: Новые зависимости в pyproject.toml
Реализовано. Добавлены `litellm`, `playwright`, `beautifulsoup4`, `lxml` с mypy overrides.

### Must Have 2: CI/CD — установка Playwright
Реализовано. Добавлен шаг `playwright install chromium --with-deps` в ci.yml и deploy.yml с кэшированием браузеров.

### Must Have 3: AI-сервис — модуль `src/chatfilter/ai/`
Реализовано. `service.py` — обёртка над LiteLLM для вызовов OpenRouter. `models.py` — Pydantic-конфигурация (модель, fallback-список, API key). `billing.py` — списание с атомарной проверкой баланса. Конфигурация через app_settings.

### Must Have 4: Баланс пользователей
Реализовано. Alembic-миграции для `ai_balance_usd` (DEFAULT 1.0) и таблицы `ai_transactions` (id, user_id, amount, balance_after, type, model, tokens_in, tokens_out, description, created_at). Стартовый баланс $1.00, проверка перед запросом, логирование транзакций.

### Must Have 5: Админка — управление AI
Реализовано. Настройки OpenRouter (API key, модель, fallback-модели). Напротив каждого пользователя: текущий баланс + кнопка/поле для начисления USD.

### Must Have 6: Профиль — история транзакций
Реализовано. Таблица транзакций AI на странице профиля с датой, типом, моделью, токенами, стоимостью и балансом после операции.

### Must Have 7: Редизайн хедера
Реализовано. Убраны inline-стили. Логическая группировка: лого + навигация слева, AI-баланс + тема/язык (иконки-кнопки) + профиль-dropdown (username, профиль, logout) справа. Hover-эффекты, spacing.

## Не выполнено

Все Must Have требования из SPEC.md выполнены.

Известная регрессия: перекрытие элементов хедера на мобильных (375px) — задача ChatFilter-3xu переоткрыта для следующей итерации.

Все 7 Must Have требований реализованы и протестированы. AI-платформа функциональна: интеграция с OpenRouter через LiteLLM, биллинг с атомарными транзакциями, управление через админку. Хедер переработан с логической группировкой элементов. CI проходит. Одна мобильная регрессия вынесена в backlog.
