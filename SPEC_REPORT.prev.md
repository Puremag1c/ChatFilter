# Отчёт по итерации

**Версия:** 0.25.1
**Дата:** 2026-03-21

## Выполнено

### Must Have 1: Единый источник правды для всех переводов (серверных и клиентских)
Реализовано. Все переводы (серверные и клиентские) хранятся в `.po` каталогах Babel. JS-переводы генерируются из `.po` через модуль `js_translations` и инжектируются inline в `window.__i18n__` в base template. Отдельные JSON файлы (`en.json`, `ru.json`) удалены.

### Must Have 2: JS получает переводы из того же источника что и сервер
Реализовано. Класс `I18n` в `i18n.js` рефакторён для чтения данных из `window.__i18n__` (inline JSON в base template). HTTP fetch удалён. Cache-busting добавлен для JS скриптов.

### Must Have 3: Все строки UI переведены на EN и RU
Реализовано. Все строки (кнопки, поля, ошибки, статусы, тосты) переведены. JS ключи замаплены на gettext msgid через модуль маппинга. SSE endpoints и API sessions исправлены для корректного locale context.

### Must Have 4: Автоматический тест полноты переводов (CI-ready)
Реализовано. `test_i18n_completeness.py` проверяет что все ключи имеют перевод на все языки. `polib` добавлен в dev-зависимости. Conftest fixture автокомпилирует `.mo` файлы.

### Must Have 5: При переключении языка — весь интерфейс на выбранном языке
Реализовано. Language switcher обновлён на reload-based flow с визуальной обратной связью. SSE endpoints передают locale. Inline переводы корректно рендерятся для обоих языков.

### Must Have 6: Существующие тесты продолжают проходить
Реализовано. Все 2224 теста проходят (было 2192, добавлены новые регрессионные тесты для SSE locale fix и i18n completeness).

## Не выполнено

Нет — все 6 Must Have требований реализованы.

### Nice to Have
- **Простота добавления нового языка**: Реализовано де-факто — достаточно добавить `.po` файл и скомпилировать `.mo`.
- **Команда/скрипт для проверки полноты переводов**: Реализовано через `test_i18n_completeness.py` (запускается через pytest).

## Итог

Система i18n ChatFilter полностью унифицирована. Двойная система (gettext + JSON) заменена на единый источник правды в `.po` каталогах. JS переводы инжектируются inline, отдельные файлы удалены. Все 6 Must Have требований выполнены, оба Nice to Have реализованы. Тесты: 2224 — все проходят. Безопасность: XSS-защита для inline JSON и переведённых строк.
