# Отчёт по итерации

**Версия:** 0.33.0
**Дата:** 2026-04-06

## Выполнено

### Must Have 1: Аудит и рефакторинг автообновления
Реализовано. Проведён аудит SSE + HTTP polling + morphdom fallback. Исправлен баг: новая карточка не появлялась после collect (добавлен refreshGroups). Исправлен race condition morphdom, удалявший карточку при переходе scraping→pending. Исправлена логика skip в morphdom, блокировавшая переход навсегда.

### Must Have 2: UI при сборе — только прогресс
Реализовано. При статусе `scraping` скрыты все кнопки (Настроить анализ, Удалить, Начать анализ). Отображается только прогресс сбора с информацией по площадкам. Кнопка «Собрать» блокируется + спиннер при отправке.

### Must Have 3: Feedback после завершения сбора
Реализовано. Toast-уведомление по завершении: «Найдено X чатов с Y площадок» или «Не удалось найти чаты». Специальное состояние карточки при 0 чатов (вместо молчаливого PENDING). Предупреждение при AI fallback.

### Must Have 4: Биллинг search-транзакций
Реализовано. Исправлен мультипликатор в settle() (расхождение с reserve). Search-транзакция теперь показывает площадки и найденные чаты вместо model/tokens. Stub-платформы (nicegram, google_search) больше не считаются успешными.

## Дополнительно (не из SPEC)

- **[Security]** TOCTOU: reserve() перемещён до create_task() в collect endpoint
- **[Security]** Неатомарный topup: заменён read-modify-write на SQL INCREMENT
- **[Reliability]** Сброс залипших SCRAPING-групп при старте сервера
- **[UX]** Блокировка кнопки сбора + спиннер во время отправки

## Не выполнено

- **Nice to Have:** Кнопка «Отменить сбор» — не реализована
- **Nice to Have:** Retry для упавших площадок — не реализован
- **Nice to Have:** Валидация модели в настройках (litellm) — не реализована

## Итог

Все 4 Must Have требования из SPEC.md выполнены. Scraping flow стал надёжным: автообновление переживает DOM-свопы, пользователь видит прогресс и результат сбора, биллинг корректно считает и отображает стоимость. Дополнительно исправлены 2 уязвимости безопасности и добавлен recovery для залипших групп.
