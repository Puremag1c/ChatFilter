# Отчёт по итерации

**Версия:** 0.26.1

## Выполнено

### Must Have 1: Логин-форма — тёмная тема
Реализовано. login.html рефакторнут на base_minimal.html — CSS-переменные тёмной темы теперь применяются. Дополнительно исправлен баг с --card-bg, который не определялся в dark theme.

### Must Have 2: Логин-форма — переключатель темы и языка
Реализовано. Переключатели темы и языка добавлены на страницу логина через base_minimal.html (как на остальных страницах).

### Must Have 3: Логин-форма — логотип
Реализовано. Логотип отображается на странице логина через base_minimal.html.

### Must Have 4: Спам "Updated proxy in pool" в логах
Реализовано. Уровень логирования понижен с INFO до DEBUG. Спам в логах устранён.

### Must Have 5: Пароль админа не виден при старте
Реализовано. Пароль дублируется через logger.warning, попадает в лог-файл. Добавлено напоминание сменить пароль.
