# Отчёт по итерации

**Версия:** 0.30.1
**Дата:** 2026-04-02

## Выполнено

### Must Have 1: Миграция telegram_id для ранних чатов
Реализовано. Alembic миграция 006 извлекает telegram_id из поля id (SUBSTR(id, 5)) для записей с telegram_id = 0 и id LIKE '-100%'.

### Must Have 2: Миграция last_check для ранних чатов
Реализовано. В той же миграции 006 заполняет last_check из created_at для записей с last_check IS NULL.

### Must Have 3: Экспорт каталога в CSV
Реализовано. Эндпоинт /api/catalog/export возвращает CSV с текущими фильтрами, сортировкой и поиском (все страницы, без пагинации). Кнопка на странице каталога. UX-улучшения: состояние «exporting...» для предотвращения двойного клика, кнопка отключена при пустых результатах фильтрации. Ячейки санитизированы от formula injection.

## Не выполнено

Все Must Have требования из SPEC.md выполнены. Nice to Have отсутствовали.

## Итог

Итерация завершена полностью. Добавлен экспорт в CSV с защитой от formula injection и UX-улучшениями. Миграция Alembic дозаполняет telegram_id и last_check для ранних записей каталога. Все тесты (backend, API, functional, visual, CI) пройдены успешно.
